# Yu Stream
Light Video LiveStreaming Service for Amazon IVS

## requirement
- PHP7
- composer
- mysql or sqlite

## install
1. clone this repository

```
$ git clone git@github.com:clom/yu-streaming.git
```

2. install library for composer
```
$ composer intall
```

3. copy .env file and settings

```
$ cp .env.example .env
```

4. migration

```
$ php artisan migrate
```
